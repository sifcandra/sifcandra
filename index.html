Tentang Saya

Saya sedang menempuh pendidikan S1 PJJ Informatika dengan fokus utama pada pengembangan kompetensi di bidang Network & Security serta Artificial Intelligence. Ketertarikan saya pada dunia jaringan komputer dimulai sejak SMK, dan terus berkembang seiring pemahaman saya tentang pentingnya keamanan sistem dan peran teknologi AI dalam memecahkan masalah kompleks di dunia nyata.

Dalam bidang jaringan, saya aktif mempelajari berbagai topik seperti konfigurasi router dan switch, pemantauan lalu lintas jaringan, serta penerapan dasar keamanan seperti firewall, VPN, dan autentikasi dua faktor. Saya juga telah mencoba berbagai tools seperti MikroTik, Cisco Packet Tracer, dan Wireshark untuk memahami cara kerja jaringan secara praktis.

Di sisi lain, saya juga mulai mendalami Artificial Intelligence, khususnya Machine Learning dan Deep Learning. Saya mempelajari dasar-dasarnya melalui kursus online dan pelatihan bersertifikat, serta membangun model-model sederhana menggunakan Python dengan pustaka seperti Scikit-Learn, TensorFlow, dan Keras. Beberapa proyek pribadi yang saya kerjakan melibatkan analisis data, klasifikasi, dan eksplorasi algoritma pembelajaran.

Belajar secara mandiri dan fleksibel melalui program PJJ memberi saya kesempatan untuk mengeksplorasi bidang-bidang ini dengan ritme yang sesuai. Saya percaya bahwa proses belajar yang berkelanjutan, dibarengi dengan latihan dan eksperimen, akan membentuk fondasi yang kuat untuk berkarier di dunia teknologi yang dinamis dan menantang ini.

Saya antusias untuk terus berkembang sebagai individu yang adaptif, ingin tahu, dan siap berkontribusi dalam membangun solusi digital yang aman, efisien, dan berdampak positif bagi masyarakat.

Proyek Pilihan

Simulasi Jaringan Juniper

Membangun topologi jaringan menggunakan perangkat virtual Juniper di GNS3. Konfigurasi mencakup routing OSPF, VLAN, dan pengujian konektivitas antar subnet sebagai latihan manajemen jaringan tingkat lanjut.

Simulasi Keamanan Jaringan & Brute Force

Melakukan simulasi serangan brute force menggunakan Kali Linux (Hydra) terhadap layanan SSH dan FTP pada jaringan lokal. Diterapkan mekanisme pertahanan seperti fail2ban dan konfigurasi firewall untuk memblokir IP penyerang secara otomatis.

Model Klasifikasi Sederhana dengan Machine Learning

Membangun model klasifikasi menggunakan Python dan Scikit-Learn dengan dataset Iris. Meliputi preprocessing data, pelatihan model dengan algoritma Decision Tree dan KNN, serta evaluasi akurasi model menggunakan confusion matrix dan visualisasi hasil.
